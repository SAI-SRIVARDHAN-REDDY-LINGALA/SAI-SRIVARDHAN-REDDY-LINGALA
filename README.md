## 2. About Section

I am an entry-level **VLSI / Digital ASIC Design & Verification Engineer** candidate with a strong foundation in **Digital Logic Design, Computer Architecture, RTL design, functional verification, FPGA prototyping, and hardware-aware thinking**. My interest is in building real digital systems that move cleanly from architecture to RTL to implementation.

My core mindset is silicon-oriented: I think in terms of **microarchitecture, timing, PPA trade-offs, reset/clock behavior, pipeline balance, modular verification, and hardware correctness**. I enjoy working on designs where the logic is not only functional, but also synthesizable, testable, and practical for FPGA-to-ASIC transitions.

My strongest areas include:
- **Digital Logic Design**
- **Computer Architecture**
- **RTL Design and Microarchitecture**
- **Functional Verification**
- **Pipelining and Timing Awareness**
- **FPGA Implementation**
- **ASIC Flow Curiosity**
- **PPA-minded hardware design**

I work with a mix of **Verilog, SystemVerilog, VHDL, MATLAB, Python, Tcl, Bash, Cadence, Synopsys, Vivado, and simulation/debug workflows**. My learning path has been shaped by hands-on laboratory work, communication-system hardware, FPGA protocol stacks, DSP architecture modeling, and CMOS/VLSI design experiments.

**Open To**
- Internships
- Entry-level RTL Design roles
- Entry-level Design Verification roles
- FPGA / ASIC implementation work
- Hardware architecture and protocol design opportunities
